## Setup Project

clone project ini
setup .env dengan isi database sipdama atau jadikan .env.example menjadi .env
buat database pada phpmyadmin dengan nama sipdama
jalankan perintah:
-   composer i
-   npm install
-   php artisan migrate
-   npm run build
-   php artisan serve
